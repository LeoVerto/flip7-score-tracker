# Flip 7 Score Tracker (Cloudflare Worker)

Track your Flip 7 scores quickly and make them available for other players live.

Thrown together by Claude Fable for a game night.

Try this at https://flip7-tracker.ballistic-tile.workers.dev

## Deploy

Deploy without an account (temporary, 60 min unless claimed):

    npx wrangler@latest deploy --temporary

Wrangler prints the live *.workers.dev URL and a claim link.
Open the claim link and sign in/up to keep the deployment permanently.

Requires Node.js 20+.
